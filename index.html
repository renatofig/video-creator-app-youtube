Como Obter as Chaves de API
1. Chave da API do Gemini
Para evitar os limites do plano gratuito (ex: 50 requisições/dia), é altamente recomendado habilitar o faturamento no seu projeto do Google Cloud.
  1. Acesse o Google Cloud Billing e configure uma conta de faturamento.
  2. Vincule seu projeto a esta conta de faturamento.
  3. Acesse o Google AI Studio.
  4. Clique em "Create API key" no projeto correto.
  5. Copie a chave gerada e cole no campo "Gemini AI:".
2. Chave de API do Google Cloud e IDs de Pesquisa
IMPORTANTE: Para separar as notícias, você precisará criar dois Mecanismos de Pesquisa Programáveis, um para cada tema.
Parte A: Chave da API (para Voz e Pesquisa)
  1. Acesse o Google Cloud Console e selecione seu projeto.
  2. No menu de navegação, vá para APIs e Serviços > Biblioteca. Procure e ATIVE as seguintes duas APIs: Custom Search API e Cloud Text-to-Speech API.
  3. Depois de ativadas, vá para APIs e Serviços > Credenciais e clique em + CRIAR CREDENCIAIS > Chave de API.
  4. Copie a chave gerada e cole no campo "Google Cloud API Key:". Esta chave única servirá para a geração de voz e para os mecanismos de pesquisa.
Parte B: IDs dos Mecanismos de Pesquisa (CX)
  1. Acesse o Mecanismo de Pesquisa Programável.
  2. Para Futebol (Tema 1): Crie um mecanismo e em "Sites para pesquisar", adicione sites de esporte (Ex: ge.globo.com/futebol/*, espn.com/soccer/*). Copie o ID gerado para o campo "ID Pesquisa Futebol (Tema 1) (CX)".
  3. Para Pescados (Tema 2): Crie um segundo mecanismo e em "Sites para pesquisar", adicione sites de agronegócio, economia e alimentos (Ex: g1.globo.com/economia/agronegocios/*, seafoodsource.com/*). Copie o novo ID gerado para o campo "ID Pesquisa Pescados (Tema 2) (CX)".

            
Parte C: Bucket do Google Cloud Storage (Obrigatório para áudios longos)

            
Para transcrever áudios ou vídeos com mais de 60 segundos, a API de Voz do Google exige que o arquivo seja enviado para um "bucket" de armazenamento. Este é um passo único.

            
    
                
  1. No Google Cloud Console, procure por Cloud Storage e clique em Buckets.
    2. 
                
  2. Clique em + CRIAR. Dê um nome único global ao seu bucket (ex: audios-transcricao-meuprojeto123). Anote este nome.
    3. 
                
  3. Escolha a localização mais próxima de você (ex: southamerica-east1 para São Paulo).
    4. 
                
  4. Mantenha as outras opções como padrão (Standard, Uniform) e clique em CRIAR.
    5. 
                
  5. Copie o nome exato que você deu ao bucket e cole no campo "Nome do Bucket Google Cloud Storage:" no aplicativo.
    6. 
            

            
3. Credenciais do YouTube (Client ID e Secret)
Atenção: a funcionalidade de verificação de duplicidade de vídeos requer a permissão de "leitura" do seu canal. Ao fazer login pela primeira vez com esta versão, você precisará re-autorizar o acesso na tela do Google.
  1. No mesmo projeto do Google Cloud, vá para APIs e Serviços > Biblioteca, procure por YouTube Data API v3 e clique em ATIVAR.
  2. Vá para APIs e Serviços > Tela de permissão OAuth. Selecione Externo, preencha os dados e na etapa "Usuários de teste", clique em + ADD USERS e adicione o seu próprio e-mail do Google.
  3. Agora vá para APIs e Serviços > Credenciais. Clique em + CRIAR CREDENCIAIS > ID do cliente OAuth.
  4. Em "Tipo de aplicativo", selecione App para computador.
  5. Após criar, copie o Client ID e Client Secret para os campos do aplicativo.

7. Como funcionam as Vinhetas e Finalizações

                
A seção "Branding com Logo" permite criar introduções e finalizações de forma flexível. A ordem de montagem do vídeo final é:

                
    
                    
  1. 1. Introdução da Pasta: Um vídeo temático da sua "Pasta de Introduções".
    2. 
                    
  2. 2. Vinheta com Logo: A vinheta criada na seção "Branding", que vem logo após a introdução da pasta.
    3. 
                    
  3. 3. Corpo do Vídeo: O conteúdo principal gerado.
    4. 
                    
  4. 4. Finalização com Logo: Um clipe final com sua marca.
    5. 
                


                
Configurando a "Vinheta com Logo"

                
Você tem três opções principais ao marcar "Adicionar Vinheta com Logo":

                
    
                    
  • Vídeo + Logo (Recomendado): Selecione um vídeo em "Vinheta Principal" e um PNG em "Logo para Sobrepor". O aplicativo irá animar seu logo por cima do seu vídeo. O áudio original do vídeo será mantido.
    • 
                    
  • Apenas Vinheta (Vídeo): Selecione um vídeo em "Vinheta Principal" e deixe o campo "Logo para Sobrepor" em branco. O aplicativo usará seu vídeo como uma vinheta simples, sem animação de logo.
    • 
                    
  • Apenas Logo (Imagem): Deixe "Vinheta Principal" em branco e selecione um PNG em "Logo para Sobrepor". O aplicativo criará um fundo animado (baseado no conteúdo do vídeo) e aplicará seu logo sobre ele.
    • 
                


                
Configurando a "Finalização com Logo"

                
Esta opção funciona de forma mais simples. Ela sempre precisa de uma imagem (PNG) no campo "Logo para Sobrepor". Ela irá gerar um fundo animado (baseado no conteúdo do vídeo) e aplicar seu logo com a animação de saída escolhida. O campo "Vinheta Principal" é ignorado para a finalização.
